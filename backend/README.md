# Social Media Project

## Overview

This project is a social media application that allows users to create, manage, and interact with posts, comments, and user profiles. It is built using TypeScript and follows a modular architecture.

## Features

- User authentication and authorization
- Create, read, update, and delete posts
- Comment on posts
- User profile management
- Error handling middleware
- Utility functions for common tasks

## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/Omartalat/social-media-project.git
   ```

2. Navigate to the project directory:

   ```bash
   cd social-media-project
   ```

3. Install the dependencies:

   ```bash
   npm install
   ```

## Usage

1. Create a `.env` file based on the `.env.example` file and configure your environment variables.
2. Start the application:

   ```bash
   npm start
   ```

## Testing

To run the tests, use the following command:

```bash
npm test
```
